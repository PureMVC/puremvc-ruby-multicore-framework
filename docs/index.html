PureMVC Ruby MultiCore Framework

PureMVC is a lightweight framework for creating applications based upon the classic Model-View-Controller design meta-pattern. It supports modular programming through the use of Multiton Core actors instead of the Singletons used in the Standard Version.

Development

Install Dependencies

bundle install

Lint Code

bundle exec rubocop

Generate RBS Signatures

rbs prototype rb src/**/*.rb --out-dir=sig

Run Type Checking

steep check

Test

ruby -Itest -e 'Dir["test/**/*_test.rb"].each { |file| require_relative file }'
RubyMine: Test file name mask: **/{*_test,test_*,*_spec}.rb

Generate Documentation

yard doc src/**/*.rb --output-dir docs --protected --private
open docs/index.html

Publish

1. Setup RubyGems API Key

mkdir -p ~/.gem
cat > ~/.gem/credentials <<EOF
---
:rubygems_api_key: API_KEY
EOF
chmod 0600 ~/.gem/credentials

2. Build the gem

gem build puremvc.gemspec
gem push puremvc-1.0.0.gem
